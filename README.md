# Property_Rental_Data

## Project Overview

- This project focuses on analyzing rental property listings dataset from Bangladesh.

- The goal is to clean the dataset, handle missing values, standardize column formats, categorize key variables, and extract meaningful insights using Exploratory Data Analysis (EDA) and visualizations.

### This project helps to understand:

- Which property features influence rental prices?

- Trends in property size, floors, furnishing, and location

- Price categorization and distribution across regions

## Key Project Tasks

- Data Loading and Inspection
- Handling Missing & Duplicate Values
- Feature Cleaning and Transformation
- Outlier Detection and Removal
- Converting BDT to INR & Country Standardization
- Creating Categorical Features
- Statistical Analysis & Visualizations
- Business Insights & Recommendations

## Insights Summary

- Larger and better-furnished properties have higher rental prices

- Majority of listings fall under mid-range pricing

- Prime locations show significantly expensive rents

- Data contained inconsistencies which required significant cleaning
